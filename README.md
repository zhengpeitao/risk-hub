## 一、介绍

### 1.1 风控业务范畴
- 合规风险：
  - 风险类型：涉黄、涉政、暴恐、不良、辱骂、广告、诈骗
  - 风险载体：文本、图片、语言、视频
- 移动风险：破解应用、外挂辅助、验证打码、违规采集
- 业务风险：
  - 账号安全：虚假注册、垃圾注册、撞库、盗号、养号
  - 交易安全：盗卡盗刷、刷单套现、恶意退款、恶意拒付、电信诈骗
  - 营销活动：羊毛党、套利、刷榜刷单
